VSTENO EDITOR: CLASSES (Overview)

Keyboard and mouse commands:

  • leftclick: in drawing area: select knot or insert knot (after selected knot) / on sliders: set tension
  • leftclick and drag: move knot or connection point
  • d + leftclick: delete knot
  • h/o/p: toggle between horizontal, orthogonal and proportional knot
  • t: toggle between locked (only 1 tension) und unlocked (separate tensions for left/middle/right)
  • f: toggle between filled shape and shape with only outer lines visible
  • l/m/r: select left/middle/right tension for tension sliders (only if tensions are unlocked)
  • arrow left/right: select preceeding/following knot
  • c: toggle on/off connection with preceeding/following points
  • +: add a new parallel rotating axis (popup window to enter shiftX)
  • -: delete actually selected rotating axis
  • ctrl+arrowleft/right: select preceeding/following rotating axis

colors and shapes:

  • green: rotating axis
  • black: connecting points
  • blue: entry knots
  • purple: pivot knot
  • red: normal knot
  • yellow border = selected knot/point
  • circle: horizontal knot (= stays on horizontal line when rotating axis moves)
  • rectangle: orthogonal knot (= follows rotating axis orthogonally)
  • star: proportional knot (= like orthogonal, but in addition, vectors are adapted proportionally + possibility to define a parallel rotating axis
